# AI Article Summarizer

A full-stack web application that allows users to input articles, and get sent back an output with details relating to that article! More features to come as well. 

## Tech Stack

- **Backend:** Node.js, Express.js  
- **Database:** MongoDB with Mongoose  
- **Authentication:** Passport.js
- **Frontend:** EJS templating, HTML, CSS, and ReactJS
